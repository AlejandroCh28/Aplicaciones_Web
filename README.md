# Essenza Luxe

Aplicación web tipo marketplace enfocada en la venta de perfumes.

---

##  Estructura del proyecto

```
Essenza-Luxe/
├── backend/    → API (Node.js + Express + MongoDB)
├── frontend/   → Interfaz (React + Vite + Tailwind)
├── docs/       → Documentación del proyecto
```

---

##  Requisitos

* Node.js (v18 o superior)
* npm

---

##  Instalación

Clonar el repositorio:

```
git clone https://github.com/AlejandroCh28/Aplicaciones_Web.git
cd Aplicaciones_Web
```

---

##  Backend

Entrar a la carpeta:

```
cd backend
```

Instalar dependencias:

```
npm install
```

Crear archivo `.env` basado en `.env.example`:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

Ejecutar servidor:

```
npm run dev
```

---

##  Frontend

Abrir otra terminal y entrar a:

```
cd frontend
```

Instalar dependencias:

```
npm install
```

Ejecutar proyecto:

```
npm run dev
```

Abrir en navegador:

```
http://localhost:5173
```

---

##  Notas

* No subir el archivo `.env`
* Usar `.env.example` como referencia
* No modificar la estructura del proyecto sin acordarlo con el equipo
